Project Name: WorldSurf

Project Description

Hi, I'm Daniel and this is my landing page for my Web Interface and Usability projects. The following are links to usability tests and their results. Throughout the project we will be creating wireframes for various elements of both a mobile app and related desktop site for a made up client, WorldSurf. The first test is a Nav Flow test for a user sign up form. The form walks a user through the steps needed to create an account. The second test explores some of the apps features and member area. Test number three is a design for the homepage of the app. Various design patterns were used in creating the wireframes to maximize usability. Tests 5 and 6 adapt the mobile screens to a full desktop site. Explore the home page and details for a featured beach.
